## 1.数据库基础

### 介绍

*   持久化数据的一种介质，用来存储和管理数据的仓库
*   持久化
    *   把数据保存到可掉电设备中以供之后使用

### 1.特点

* 将数据放到表中，表再放库中
* 基于客户机 — 服务器的 DBMS ( C/S  Mysql oracal sqlserver)

### 2.英文

* DB

    *   数据库（database）
    *   存储数据的仓库，保存一系列有组织的数据

* DBMS

    *   数据库管理系统（database management system）
    *   数据库是通过DBMS创建和操作的容器
    *   数据库软件或数据库产品，
    *   分类

* SQL

    *   结构化查询语言（structure query language）
    *   专门用来与数据库通信的语言 ，不是某个数据库软件特有的，几乎所有的主流数据库通用的语言
