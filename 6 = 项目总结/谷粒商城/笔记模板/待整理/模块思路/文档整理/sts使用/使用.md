## 父子平行项目创建

```java
new -- new maven project -- 简单安装勾选  -- 开始创建父项目 parent（看父工程配置）
parent 项目上点击 -- new -- maven module
    webUI(war)   // 注意这个打包为 war 会出错，建成以后： 右键 -- javaee tools -- generate deployment... 即可
    entity(jar)
    component(jar)
    // 下面两个是相同的, 注意看下 parent 对不对

平行项目创建
parent
    右键 -- new maven project (workspace location 默认就行了) -- 添加进 两个id 即可


引用加入（依赖关系添加）
    // 推荐使用 图形化界面添加
    加入 a 依赖 b : a 要使用 b 中的某项功能
    点击 a  的 pom.xml --> 找到 dependencies -- add --弹出框中间位置搜索 b -- 选中然后 ok 
    // 记得 ctrl s 保存！！！！！





简单安装
    Create a simple project (skip archetype selection)

```
