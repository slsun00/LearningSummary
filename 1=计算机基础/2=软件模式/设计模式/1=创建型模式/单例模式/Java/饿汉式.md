## 静态常量

### 重要理解

```java
主要理解
    static final
    怎么实现线程安全？？不会多线程同时访问吗？
    
```



### 实现步骤

* 构造器私有化
* 类内部创建对象
* 向外暴露一个静态的公共方法
* 实现

### 例子

```java
package cn.it.day02;

public class Single {
    public static void main(String[] args) {
        //
        Singleton instance = Singleton.getInstance();
        Singleton instance2 = Singleton.getInstance();
        System.out.println(instance2 == instance);
    }
}

class Singleton {
    // 构造器私有化， 不能够 new 
    private Singleton(){

    }

    // 成员属性
    private static final Singleton  instance = new Singleton();

    //3. 提供一个静态方法 ， 返回实例
    public static final Singleton getInstance() {

        return instance;
    }
}
```





## 静态代码块

```java
package cn.it.day02;

public class Single {
    public static void main(String[] args) {
        //
        Singleton instance = Singleton.getInstance();
        Singleton instance2 = Singleton.getInstance();
        System.out.println(instance2 == instance);
    }
}

class Singleton {
    // 构造器私有化， 不能够 new 
    private Singleton(){

    }

    // 成员属性 , 本类内部创建对象实例
    private static final Singleton;
    // 实例对象化
    static {
         instance = new Singleton();
    }

    //3. 提供一个静态方法 ， 返回实例
    public static final Singleton getInstance() {

        return instance;
    }
}
```
