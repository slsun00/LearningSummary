### 介绍

* 直接创建好对象，这样不需要判断为空，同时也是线程安全
* 在导入包的同时会创建该对象，并持续占有在内存中




```go
type Tool struct{
    value int
}

// 声明私有变量作为单例,进行初始化
var instance = &Tool{666}

func GetInstance() *Tool {
    return instance
}

// 使用 init 函数,私有变量单例，进行初始化
// 从代码中可以看到 instance 这个人有多饥饿了吧？在属性中它便把自己实例化了一次，也就是在类加载的同时，它就开始找食物并且吃掉它。
var instance *Tool
func init(){
    instance = new(Tool)
    instace.value = 666
}


```
