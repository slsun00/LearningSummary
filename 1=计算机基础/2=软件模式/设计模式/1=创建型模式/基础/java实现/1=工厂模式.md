#### 手机

```java
public interface Phone {
    void call();
}

public class HuaPhone implements Phone {
    @Override
    public void call() {
        System.out.println("华为手机");
    }
}

public class MiPhone implements Phone {
    @Override
    public void call() {
        System.out.println("小米手机");
    }
}
```

### 工厂模式：一类一例

```java
// 一个类中可以创建一个对象： 一个工厂可以生产一种产品
public class PhoneFactory {
    public Phone Create(String phoneName) {
        switch (phoneName) {
            case "huawei" :
                return new HuaPhone();
                break;
            case "xiaomi":
                return new MiPhone();
        }
        
    }
}

```



### 抽象工厂 ：一类多例

```java
public interface Factory {
    public Phone createPhone();
    public Watch createWatch();
}

// 一个类中可以创建多个对象： 一个工厂可以生产多种产品
public class MiFactory implements Factory {
    @Override
    public Phone createPhone() {
        return new MiPhone();
    }

    @Override
    public Watch createWatch() {
        return new MiWatch();
    }
}

// 测试使用
public class Main {
    public static void main(String[] args) {
        Factory mi_factory = new MiFactory();
        Phone miphone = mi_factory.createPhone();
        miphone.call();
        Watch miwatch = mi_factory.createWatch();
        miwatch.ring();
    }
}
```
