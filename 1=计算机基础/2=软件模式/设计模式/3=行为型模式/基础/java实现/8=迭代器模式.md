### 迭代器

```java
// interface 
public interface StudentItrator {
    boolean hasNext();
    Student next(); // 
}

// clsaa
public class StudentItratorImpl implements StudentItrator {
    private List<Student> list;
    private int position = 0;
    
    // 传入一个 容器
    public StudentItratorImpl(List<Student> list) {
        this.list = list;
    }
    
    @Override
    public boolean hasNext() {
        return position < list.size();
    }
    
    @Override
    public student next() {
        // student = list.get(position)
        // position++
        // return student
        return list.get(position++);
    }
}
```
