## 顺序栈

### 结构

```java
package com.it.puke.queue;



class Stack {
    private int top;
    private int maxSize;
    private int[] arr;

    public Que(int maxSize) {
        this.top = -1;  // 栈顶： 初始化的时候， top 指向栈底的 前一个位置，即 -1
        this.maxSize = maxSize;
        this.arr = new int[maxSize];
    }

}

```



### 增 - 压栈

```java
    public void push(int data) {
          // 做最好加上 rear == font 后，让指针归位为 -1
        // 让数组能稍微利用，但是如果 真的font < rear ，就真的加不进去了
        if (isFull()) {
			throw new RuntimeException("堆栈已满")；
        }
        this.rear++;
        this.arr[rear] = data;
    }
```
