## 普通插入

```java
public class Insert{
    public void sort(int[] arr) {
        // 把第一个元素当成默认的形式
        for (int i = 1; i < arr.length; i++) {
            int temp = arr[i];
            // i-1 是有序列表最后一个元素
            for (int j = i-1; j >= 0; j--) {
                if (temp < arr[j]) {
                    arr[j++] = arr[j];
                } else if (temp >= arr[j]){
                    arr[j--] = temp;
                    break;
                }
            }
        }
    }        
}
```
