## 业务思路

```java
[2,-3,0,5,-1,8,6]    进行排序   [-3,-2,0,2,5,6,8]
    
拆分 data1 [-3,-2,0] data2[2,5,6,8]
    如果 x >= 2  则在区间 data2[2,5,6,8]
    如果 x <= 0  则在区间 data1[-3,-2,0]
    
// 假设 x = 8
拆分 data2[2,5,8,6]  为 data1 [2,5] data2[6,8]
    如果 x >= 6  则在区间  data2[6,8]
    如果 x <= 5 则在区间 data1 [2,5]
    
拆分 data2[6,8]]  为 data1 [6] data2[8]  
    如果 x >= 8  则在区间  data2[8]
    如果 x <= 6 则在区间 data1 [6]
```
