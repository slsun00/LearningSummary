

## 接口

```java
注意
    树操作都是以二叉搜索树为基础为例子的， 
    因为不知道规则，无法进行遍历等操作，也没法判断应该添加在什么地方

树是由一个根结点和若千棵子树构成。树中结点具有相同数据类型及层次关系

1. 构造空树 T 	inittree
2. 销毁树 T	destorytree
3. 若树存在，将树清空为空树	cleartree
4. 若树为空 返回 true ,否则返回 false treeempty
5. 返回树的深度 treedepth
6. 返回根节点 root
7. 返回结点的值 value
8. 给输的结点赋值 assign
9. 非根节点，返回双亲 parents
10. 非叶结点，返回其最左孩子 ，否则为空 leftchild
11.返回右兄弟	rightsibing
12. insertChild
13. deleteChild
```
