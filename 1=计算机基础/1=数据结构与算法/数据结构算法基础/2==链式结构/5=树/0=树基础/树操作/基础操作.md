## 节点比较

### 介绍

```java
基础类型： 直接比较就可以
其他类型： 需要有一个比较标准
```



### java 实现

```java 
// 比较， 初始化需要有一个包含比较器的构造函数
private Comparator<E> comparator;

// 节点比较
// 基础类型基本实现了 comparable
private int compare(E e1, E e2) {
    if (comparator != null) {
        return comparator.compare(e1,e2);
    }
    return ((Comparable<E>)e1).compareTo(e2);
}

// 注意在实现二叉搜索树的时候，需要实现 comparable 接口
```
