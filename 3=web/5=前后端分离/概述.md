## 介绍

```java
前后不分离
    通过引入的方式使用vue，也就是本文实现的方式
前后半分离
    前后端项目写在一个项目里，分别搭建工程，
    前端项目也通过tomcat部署管理，
    后端工程师使用起来比较得心应手，且前端项目拆分时也比较方便
    // 可以参考：https://github.com/xichengxml/springboot-vue-template，
前后分离。
    前后端分别搭建项目，不多介绍
```



## 不分离

```java
通过
	后端
    	只是进行数据查询操作
    	需要同时通过后端的的模板渲染引擎进行：数据和页面的结合
    前端
    	只负责展示
```



## 分离

```java
通过
	后端
    	只是进行数据查询操作
    前端
    	进行数据渲染，使用前端自己的模板
```
