## 介绍

```java
非学术的说法: 
	通常个人电脑无论是连接WIFI上网还是用网线上网，都是属于局域网里边的，外网无法直接访问到你的电脑，
    // 内网穿透可以让你的局域网中的电脑实现外网访问功能
        
作用
	通过内网穿透可以实现通过外网直接访问到你的电脑
	能够像访问www.baidu.com这类的万网似的访问本地机器        
        
做法
    1 发布应用/网站，外网可以访问(这是主要用途)；
        我们平时做的应用开发都只能在局域网本地访问，通过内网穿透，可以是全外网访问(只要有网就能访问到);
		在做微信/支付宝支付回调的时候就需要用到这个了
    2 可以实现远程控制(在家访问公司电脑不是问题)
```

## 业务

```java

在支付功能（微信或者支付宝）， 都有一个回调机制，回调的地址（需要微信访问到）需要被外网访问到
在调试的时候， 在本地调节最快捷高效， 而且本地没有调好，也不会上传到服务器
    所以： 需要让本地的 ip 地址能够被外网调用到， 这就需要内网穿透了！！

    
//=========================    
在做支付功能时，不管是微信还是支付宝，都会有一个回调机制，这里涉及的回调地址要外网能够访问到的那种，即使我们有自己的服务器，有对外的公网IP，但是如果单单只是调试这个步骤来说，还是本地调试来的快捷高效，而且本地不调好，怎么能放心提交代码，所以怎么让本地的ip地址能够被外网调用到，这就需要用到内网穿透了。
```
