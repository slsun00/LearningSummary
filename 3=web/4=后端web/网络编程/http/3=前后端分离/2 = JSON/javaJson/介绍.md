## fastjson & gson

### 介绍

```java
把Java对象JSON序列化
    Jackson速度最快，在测试中比 Gson 快接近50%， FastJSON 和 Gson 速度接近。.
 把JSON反序列化成Java对象
    FastJSON 、Jackson速度接近， Gson 速度稍慢，不过差距很小。.
测试比较 Gson 与 Fastjson 性能，
    发现数据量少的情况下 fastjson 比 Gson 稍慢， 
    数据量大的情况下 fastjson 比 Gson 稍快

```

### 使用

```java
// 在项目选型的时候可以使用Google的Gson和阿里巴巴的FastJson两种并行使用
```
