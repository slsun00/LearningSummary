## @conditional

```java
@conditional(
作用：{
   	满足指定条件，才进行组件注入
}    
位置：{
    类
	方法        
}
属性：{
    Class[] value
}    
继承：{
    // 常用
    ConditionalOnBean
    ConditionalOnMissingBean
        
    ConditionalOnClass
    ConditionalOnMissingClass
        
    ConditionalOnResource        
        
}    
)

```
