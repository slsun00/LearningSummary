

## 1. Spring Boot概述

```java
Spring Boot是一个便捷搭建 基于spring工程的脚手架；
作用是帮助开发人员快速搭建大型的spring 项目。
简化工程的配置，依赖管理；
实现开发人员把时间都集中在业务开发上。
    
使用Spring Boot创建java应用，并使用java –jar 启动它，就能得到一个生产级别的web工程。    
```

## 2. Spring Boot入门

### 步骤

```java
// Spring Boot工程可以通过添加启动器依赖和创建启动引导类实现快速创建web工程
1. 创建工程；
2. 添加依赖（启动器依赖，spring-boot-starter-web）；
3. 创建启动类；
4. 创建处理器Controller；
5. 测试
```

### 依赖配置 pom.xml

```xml
<!--springboot启动器-->
<!--
	Spring boot的项目必须要将parent设置为spring boot的parent，
	该parent包含了大量默认的配置，大大简化了我们的开发。
-->
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>1.5.2.RELEASE</version>
</parent>

<!-- web 支持-->
<!--
	父版本设置过版本了，子版本就不需要使用
-->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

### java 代码

```java
// com\itheima\Application.java
// 启动类
// Spring Boot项目通过main函数即可启动，我们需要创建一个启动类
@SpringBootApplication 
public class Application { 
    public static void main(String[] args) { 
        SpringApplication.run(Application.class, args); 
    }
}

// com\itheima\controller\HelloController.java
// controller
@RestController 
public class HelloController { 
    @GetMapping("hello") 
    public String hello(){ 
        return "hello, spring boot!"; 
    } 
}
```
