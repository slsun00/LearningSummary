## 父子项目

### 结构

```java
// 父项目只用作配置使用，不用于代码编写
parent
    maven_1  项目
    maven_2  项目
    maven_3  项目
    pom.xml
```

### 创建

#### 方式一

```java
先
    创建一个maven项目作为父项目，
然后
    删除掉父项目中的 src target 目录
    保留 其中的 pom.xml 文件
最后
    父项目文件夹上正常右键创建 maven 项目即可
```

#### 方式一

```java

```
