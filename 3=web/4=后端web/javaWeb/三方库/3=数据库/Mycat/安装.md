### 安装

```java
0. tar.gz 文件
    放到 /opt  文件目录下
1、解压后即可使用
	解压缩文件拷贝到 linux 下 /usr/local/
2、三个配置文件
    schema.xml：定义逻辑库，表、分片节点等内容
    rule.xml：定义分片规则
    server.xml：定义用户以及系统相关变量，如端口等
```
