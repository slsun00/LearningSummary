## 数据

### data

```js
属性
	对象
格式 ：
	data { 属性名 : 属性值 ， 属性名 ：属性值，...}
		属性名 ：字符串(不需要加引号)
		属性值 ：对象、函数

作用  ：
	1. Vue 实例的数据，可以是从后台传过来的
    2. 属性名的变量，放入模板中进行渲染显示

    指定初始化状态属性数据的对象
    vm也会自动拥有data中所有属性
    页面中可以直接访问使用
    数据代理: 由vm对象来代理对data中所有属性的操作(读/写)
```

### methods

```js
类型
	{[key：string] : Function}
作用
	定义属于 vue 的一些方法，可以在其他地方、指令中调用
注意：	
    包含多个方法的对象
    供页面中的事件指令来绑定回调
    回调函数默认有event参数, 但也可以指定自己的参数
    // 所有的方法由vue对象来调用, 访问data中的属性直接使用this.xxx
```

### computed

```js
计算属性
	computed：{
        // 1. 
		// 这个在使用的时候，不使用 括号，当成一个属性，直接使用
        // 执行 ：初始化显示/相关的data属性属性数据发生改变，计算并返回当前属性的值
		fullName:function(){}
        
        // 2. 
        // 完整的原理展开如下，使用属性的时候，会自动调用下面两个函数
        // 所以写的时候，完整需要事项下面两个函数
        fullName:{
            setter:function(){}  // 监视
            getter:function(){}
        }
        // 3. 
       // 一般情况下是只需要一个只读的属性
        fullName:{
            get:function(){}
            // 当读取当前属性值的时候回调，计算并返回当前属性的值
            // 回调函数，当属性值发生改变时回调，更新相关的数据
            get(value){}   // value 就是当前的最新属性值
        }
        // 4. 
        //只读属性的 简化缩写就是，所以就默认你使用的是  getter 
        fullName:function(){}
        //-------------------------------------------
        // 5. 
        // 所以 fullName 是一个对象，即使简写成函数形式，他也不是函数，使用不要加小括号
	}
// 你在使用的时候，
直接调用 getter   VueObject.fullName 
你不设置  setter  的话 直接使用 VueObject.fullName = "dd" 是无法改变其内部的值的，要定使用就要调用 setter方法进行设置新值，一般前端是不允许改的

1. 包含多个方法的对象 ，
2. 计算属性，有缓存


对状态属性进行计算返回一个新的数据, 供页面获取显示
计算属性存在缓存，多次读取只执行一次 getter 计算


利用set/get方法来实现属性数据的计算读取, 同时监视属性数据的变化
如何给对象定义get/set属性
	在创建对象时指定: get name () {return xxx} / set name (value) {}
  	对象创建之后指定: Object.defineProperty(obj, age, {get(){}, set(value){}})


计算属性
	数据显示，多个数据结合
    监听器：当属性变化时，回调函数自动调用，在函数内部进行调用
    
注意
	conputed 中的函数 跟 methods 中的缩写是一样的
    	fullName:function(){}   本质一个对象
         fullName(){}			缩写
```
