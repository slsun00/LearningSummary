### 注意

```java
axios 中的 baseURL
    baseURL + 请求相对路径   ： 拼接成完整的请求路径
webpack 中的 proxyTable
    	测试环境为了避免浏览器下的跨域访问，而以nodejs代理成同前端页面（即浏览器地址栏）同域的一种处理
        指定proxyTable后， axios就不需要指定baseUrl了
		proxyTable会把base + '/api/c/xx'代理到【base baseUrl + '/api/c/xx'】的接口地址上。	
```
