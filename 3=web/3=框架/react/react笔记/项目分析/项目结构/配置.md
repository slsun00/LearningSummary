## 添加 @ 符号

```java
1. 在webpack.config.dev.js 和 webpack.config.prod.js 文件中搜索 alias 关键词
2. 在相关对象中添加一个键值对：'@': paths.appSrc
    此时的appSrc 就是项目的src目录
3. 为了防止技术升级造成的无法匹配问题的现象，在此添加版本信息如下
    "devDependencies": {
      "react-scripts": "^1.1.4"
    },
    "dependencies": {
       "react": "^16.3.2"
    }
4. 想要导入 src/pages/home/index.js 只需要把import的路径写成 @/pages/home 就可以
```
