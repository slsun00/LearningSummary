## 样式处理

### 内联

```java
介绍
    React 推荐使用内联样式。
    我们可以使用 camelCase 语法来设置内联样式. React 会在指定元素数字后自动添加 px 
    

    var myStyle = {
        fontSize: 100,
        color: '#FF0000'
    };
    ReactDOM.render(
        <h1 style = {myStyle}>菜鸟教程</h1>,
        document.getElementById('example')
    );
    
```
