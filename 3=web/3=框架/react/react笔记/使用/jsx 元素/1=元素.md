```java
概述
	jsx js xml 的简写， 表示在 js 代码中写 xml(html) 格式的代码
    是 es 的语法扩展，需要进行 label 编译处理以后才能在浏览器中使用
    // 脚手架中通过 @bable/preset-react 进行编译 jsx
```

## 元素-- 虚拟dom

### 介绍

```java
元素
    介绍
    	元素是构成 React 应用的最小单位，JSX 就是用来声明 React 当中的元素。
        它用于描述屏幕上输出的内容
		React 当中的元素事实上是普通的对象，
     	React DOM 可以确保 浏览器 DOM 的数据内容与 React 元素保持一致。
    
    元素
    	// 虚拟 dom 
		const element = (<h1>Hello, world!</h1>)
虚拟 dom 
    1. 本质上是一个 object 类型对象
    2. 虚拟 dom 比较轻， 因为是 react 内部在使用，无需真实 dom name多属性
```



### 语法

```java
介绍
     

// jsx 创建 react 元素
const title = <h1>hello jsx</h1>   
// const title = (<h1>hello jsx</h1>)

ReactDOM.render(  
  title,
  document.getElementById('root')
);
    
```

### 命名

```java
1. 遵循驼峰命名法
2. 特殊属性名
    class -- className
	for -- htmlFor
    tabindex -- tabIndex
3.没有子节点的 react 元素可以使用 /> 结束
4. 推荐使用小括号进行包括，避免 js 自动插入分号陷阱        
```

### 渲染更新

```java
介绍
    React 元素都是不可变的。当元素被创建之后，你是无法改变其内容或属性的。
    更新界面的唯一办法是创建一个新的元素，然后将它传入 ReactDOM.render() 方法：
    

function tick() {
  const element = (
    <div>
      <h1>Hello, world!</h1>
      <h2>现在是 {new Date().toLocaleTimeString()}.</h2>
    </div>
  );
    
  ReactDOM.render(
    element,
    document.getElementById('example')
  );
}
 
setInterval(tick, 1000);

```

## js 嵌入

```java
介绍
    可以在 JSX 中使用 JavaScript 表达式。表达式写在花括号 {} 中

ReactDOM.render(
    <div>
      <h1>{1+1}</h1>
    {/*
         // 在 JSX 中不能使用 if else 语句，
        // 但可以使用 conditional (三元运算) 表达式来替代
    */}
      <h1>{i == 1 ? 'True!' : 'False'}</h1>
    </div>
    ,
    document.getElementById('example')
);
```
