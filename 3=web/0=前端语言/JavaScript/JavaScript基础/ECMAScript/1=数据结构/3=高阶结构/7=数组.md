```js
        var obj = new Array()   
    创建指定长度数组
        var obj = new Array(Size)// 创建一个长度 n 的空数组
    指定元素数组
        var obj = new Array(元素1，元素2，元素3，... ,元素N)
    
字面量创建
	空数组
    	var obj = [];  // [n] 是一个函数
    单维数组
        var obj = [元素1，元素2，元素3，... ,元素N]
    多维数组
        var obj = new Array[ [数组1],[数组2]，... ,[数组N]]
```

## 基本操作

### 存取数组

```js
单维数组
	数组名[下标索引]
多维数组
	数组名[外层数组下标][内层元素下标]
下标类型
	数值型
    非数值型
    	转化为字符串 、生成关联数组 、下标将作为对象属性的名字
特性
	1. 长度是弹性的，可以自由伸缩
    2. 下标从 0 开始
    3. 数组元素可添加到对象中
  
```

### 增加修改

```go
之前存在就是增加，之前没有就是修改
使用 [] 运算符指定下一个下标
arr[1] = 13;

// 最后一个位置添加元素
arr[arr.length] = 值
```



### 删除数组

```js
delete 数组名[下标]
```

### 遍历数组

```js
for (var 数组元素变量 in 数组)
    
 for (var i=0;i<arr.length;i++) {
     console.alert(arr[i])
 }
```
