## 日期对象

### 概述

*   Date 对象是一个内置数据类型

### 声明

```js
new Date()
参数
	no arguments 
    	不带参数，Date() 创建一个日期对象，设置为当前的如期和时间
    milliseconds
    	？？
   	datesting
    	一个日期格式的字符串，可以被 DateParse() 方法接受的格式
    arguments
    	year		 年   ，整数。应该写完全 2020
		month		 月  ， 整数  0(一月)--11（十二月）
        date		  日  ，整数  
        hour		  时   ，整数 24小时
        minute		  分
        second		  秒
        milliseconds   毫秒
```

### 属性

```JS
constructor		返回创建该对象的函数的引用
prototype		允许型对象添加属性和方法
```

### 方法

```js
Date()			返回当前的日期和时间

// 获取时间  根据本地时，从Date对象返回
getFullYear()	根据本地时，从Date对象返回以四位数返回 年份   （xxxx）
getMonth()										月份          1-11
getDate()										一个月的某一天 （1-31）
getDay()										一周的某一天	（1-6）
getHours()										小时		(0-23)
getMinutes()									分钟		(0-59)
getSeconds()    								秒		 (0-59)
getMilliseconds() 								毫秒	 (0-999)


getTime()				根据本地时，返回到1970年1月1日至今的毫秒数
getTimezoneOffset()		 返回本地时间与格林威治标准时间(GMT)的分钟差

// UTC  根据世界时，从Date对象返回
getUTCFullYear()	根据世界时，从Date对象返回 四位数年份  
getUTCMonth()								月份			0-11
getUTCDate()								月中的一天       1-31
getUTCDay()									周中的一天       0-6
getUTCHour()								小时			0-23
getUTCMinutes()								分钟			0-59
getSeconds()								秒			0-59
getMillisecond()							毫秒			0-999

// 设置时间  根据本地时，设置 Date 对象中的
setFullYear()		根据本地时，设置 Date 对象中的 年份     		四位数
setMonth()			  ...    ,       ...    	月份            1-31
setDate()			  ...    ,       ...		月中的某一天     1-31
setday()			  ...    ,       ...		周的某一天       0-6
setHours()		      ...    ,       ...		 小时		     (0-23)
setMinutes()	      ...    ,       ...	      分钟	     (0-59)
setSeconds()          ...    ,       ...		   秒	       (0-59)
setMilliseconds()      ...    ,       ...		  毫秒	     (0-999)

setTime()			根据本地时，以毫秒设置 Date 对象

// 设置 UTC   根据世界时，从Date对象返回
setUTCFullYear()	根据世界时，从Date对象返回 四位数年份  
setUTCMonth()		           ....         月份			0-11
setUTCDate()		           ....         月中的一天       1-31
setUTCDay()			           ....         周中的一天       0-6
setUTCHour()		           ....         小时			0-23
setUTCMinutes()		           ....         分钟			0-59
setSeconds()		           ....         秒			0-59
setMillisecond()	           ....         毫秒			0-999

toDateString()			把 Date 对象的日期部分转化为字符串
toLacalDateString()      根据本地时间，把 Date 对象的日期部分转化为字符串
toLocalFormat()			将一个日期转化成一个 格式化 的字符串
toLocalString()			根据本地时间格式，把 Date 对象转化为字符串
toLocalTimeString()		 根据本地时间格式，把 Date 对象的时间部分转化为字符串


toSource()			返回一个字符串，代表一个等价的日期对象的源码
toString()			Date 对象转化为字符串
toTimeString()		Date 对象的时间部分转化为 字符串
toUCTString()	    根据世界时间，把 Date 对象转化为 字符串
valueOf()		    返回 Date 对象的原始值
```

*   静态方法

    ```js
    Date.parse()		返回 1970年1月1日午夜到指定时间（字符串）的毫秒数
    Date.UTC()			根据世界时，返回  1970年1月1日 到指定日期的毫秒数
    ```
