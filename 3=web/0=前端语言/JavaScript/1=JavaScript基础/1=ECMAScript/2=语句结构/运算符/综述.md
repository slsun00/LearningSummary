## 三点运算符(es6)

### 介绍

```java
三点运算符
    将一个变量迭代分割, 逗号隔开的值序列组合成一个数组
场景分类    
    剩余运算符（spread ）
    	// 函数的参数 ： 值 =》 数组
    	// 赋值的一方：等号右边，或者放在实参上
    	把逗号隔开的值序列组合成一个数组,使用在中
    扩展运算符（rest）
    	// 在任何地方, 数组 =》 值
    	// 被赋值的一方： (等号左边、放在形参上)
    	把数组或类数组对象展开成一系列用逗号隔开的值
本质
	1. 会调用默认的 Iterator 接口， 可以将任何部署了 Iterator 接口的数据结构，转为数组        
        var str = 'hello'
        [...str]   //等价于  ['h','e','l','l','o']
```



### 剩余运算符

```java
// rest(可变)参数 ，用来取代 arguments , 但是比 argument 灵活，只能是最后部分形参参数
// ...value 中的 value 是一个真数组，可以使用 数组的方法
function foo(a,...value) {
    console.log(value)		// 这个是最后部分的参数
    consolse.log(arguments)  // 这个是全部参数
}
foo(3,4,5,6)  //  这里 value 就是 数组 4,5,6
```



### 扩展运算符

```java
扩展运算符： {
介绍： {    
    将一个数组转为用逗号分隔的参数序列
        
例子： {
	var arr = [1, 2, 3]        
	var arr2 = [...arr, 4, 5] 
	等价于 =》 var arr2 = [1, 2, 3, 4, 5]    
扩展

	2. 由于扩展运算符可以展开数组，所以可以用来替代函数的 apply 方法        
            function f(x, y, z) {
              // ...
            }
            // ES5 的写法        
            var args = [0, 1, 2];
            f.apply(null, args);

            // ES6 的写法
            let args = [0, 1, 2];
            f(...args);

```
