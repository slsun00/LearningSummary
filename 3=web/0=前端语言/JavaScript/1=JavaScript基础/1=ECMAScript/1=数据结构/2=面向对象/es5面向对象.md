##  面向对象

*   一般一个语言提供给开发者四种基本功能，就可以认为是面向对象

    ```js
    1. 封装
    2. 继承
    3. 多态
    4. 聚合
    ```

*   封装

    *   把相关信息，无论数据或者方法，都存储在一个对象中的能力

*   继承

    *   一个类依赖另一个类（另一些类）中的一些性质和方法

*   多态

    *   写一个函数或方法，这个函数或方法可以以各种不同形式工作的能力

*   聚合

    *   一个对象存储在另一个对象中的能力(兄弟之间)
