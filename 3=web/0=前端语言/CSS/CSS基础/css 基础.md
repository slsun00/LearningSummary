## 样式引入

### 样式

* 作用
  
  * 定义如何显示 HTML 元素，用来设置网页元素的样式
  
  * 到 HTML 4.0 中，是为了解决内容与表现分离的问题

* 存储位置
  
  * 存储在样式表中


```html
浏览器缺省设置
外部样式表	<link>
    完全实现结构和样式的分离，需要引入，可以控制多个页面
	
内部样式表 
	（位于 <head> 标签内部）
      结构与样式部分分离，较多使用，控制一个页面
内联样式
	权重高，但是结构样式混乱，较少使用
     （在 HTML 元素(标签)内部，控制一个标签）

因此，内联样式（在 HTML 元素内部）拥有最高的优先权，这意味着它将优先于以下的样式声明：<head> 标签中的样式声明，外部样式表中的样式声明，或者浏览器中的样式声明（缺省值）
```

### 外部样式

```css
链接式

编写
	将 css 样式编写到一个外部的 CSS 文件中，扩展名为 .css
	这个外部文件中都是 选择器{样式键值对} 的代码，会对引入的文件的响应标签起作用

引入
    <head>
    	<link rel="stylesheet" type="text/css" href="mystyle.css">
    </head>


        rel  固定，样式表
        href 样式表所在的路径

优势
	可以在所有网页中复用，提高工作效率
	可以使用浏览器的缓存机制，从而加快网页的加载速度
```

### 内部样式

```css
内部样式表，嵌入式

位置
    /*将样式编写到 head 中的 style 标签里 */
	虽然说可以随便写，但是一般写到 head 中 
格式
    <head>
        <style>
            /*选择器 p 选中的内容设置为红色，字体大小为 50px*/
            p {
                color:green;
            }  
        </style>
    </head>

问题
	只能第一个网页起作用，不能跨网页复用
```

### 行内样式

```html
<!--内联样式，行内样式、行内式-->

特殊的样式需要应用到个别元素时，直接在标签内通过设置 style 属性来设置元素样式
内联样式的方法是在相关的标签中使用样式属性。
样式属性可以包含任何 CSS 属性。

<p style="color: red; margin-left: 20px">
	This is a paragraph
</p>

缺点
	不方便后期维护，开发过程中一定不要使用内联样式，耦合度过高
	 内联样式，只对一个标签生效，如果希望影响多个元素，都复制一遍
	前往不要用

```

## 选择器(符)

```css
作用
	本剧不同需求把不同的标签选择出来，选择标签用的
```

### 类型

```css
元素选择器(符)
    <p id="nan" class="red"></p>
    标签选择器 p{}
    类 选择器 .red{}  /*css用*/
    id 选择器 #nan{}  /*id一般给js用*/
    通用选择器 *{}

关系选择符(可以深度套娃)
	包含选择器 E F  命中 E 的所有后代元素 F
	子选择器  E>F   选择父E的子元素 F
	后相邻选择器 E+F  选择 E 的 后相邻元素 F
	兄弟选择器 E~F  选择 F 的兄弟(同级别)元素 F

属性选择符
伪类选择器
	：表示，用于向某些选择器添加特殊的效果
	连接伪类  :link :visited :hover :active
		a:hover{}  鼠标从 a 标签上滑过需要添加的效果
	
伪对象选择符
```

### 选择器优先级

```css
// 权重
样式冲突
    通过不同的选择器，选中相同的元素，为相同的样式设置不同的值，此时发生样式冲突
    发生冲突有优先级（权重）决定

选择器相同
	执行行层叠性
选择器不同
	选择器优先级权重

优先级
    分组选择器单独计算 ,复合选择器单独使用
    
    无穷大  	    1000        100            10            1          0        没有优先级
   !important  >>  内联样式 > id 选择器 > 类和伪类选择器 > 元素选择器 > 统配选择器 > 继承的样式 
	(sic)
	权重叠加 ：
		不能夸数量级，id 选择器再多，也超不过内联样式
		选择器越具体，优先级越高
		优先级相同的，优先使用靠下的选择器

特殊
	a 链接 ： 浏览器默认指了一个样式，蓝色的 有下划线
```

### 文档流

```go
normal flow
    网页是一个多层的结构，一层叠一层
    通过 css 可以分别为每一层设置样式
    作为用户只能看到最上面的一层
    最底下一层，就叫文档流，文档流是网页的基础，我们所创建的元素默认都是在文档流中

元素状态
    在文档流中
    不在文档中（脱离文档）

在文档流中
    块元素
        块元素在页面中独占一行 
        默认宽度是父元素的全部（把父元素撑满）
        默认高度是被内容撑开（子元素）
    行内元素
        行内元素不会独占一行，只占自身的大小
        行内元素在页面中左向右水平排列，如果一行 中不能容纳所有的内容，则会换到第二行继续自左向右排列
        行内元素默认宽度和高度被内容撑开
```

### 行高

```html

            可以指定一个大小  px em

    行高用来设置文字的行间距    
        行间距 = 行高 - 字体大小

字体框
```

## 布局

```css
1.确认网页的版心（可视区域），可测量得知
2. 分析页面中的行模块，以及每个行模块中的列模块
3. 一行中的行模块经常浮动布局，先确定每个列的大小，之后确定列的位置
4. 制作 html 结构 ， 遵循现有结构后有样式的原则，结构永远最重要
5. 先清除结构，后写代码
```
