## jQuery 对象

### 介绍

```js
介绍
	// 每一种语言，有各自的对象，方便处理
	jQuery 函数，返回的是 jQuery 对象，
特点
	// 不能直接使用 JavaScript的方法，所以使用 $()[1].innerHTML
	// $() instanceof Array   false
	jQuery 对象是一个包含dom元素的伪数组对象
    注意: 是伪数组, 包含匹配的任意多个 dom 元素
语法
声明对象，;
	

函数对象, 当成对象使用:
     
```

### 分类

```js
声明对象
	语法
    	$xxx (执行$()得到的)
	例子
    	 var $img = $( ... )
函数对象
	语法
		// 静态方法，直接调用: 对象的直接访问                      
		 $.xxx             
	例子
		// $("img").each(callback) == $.each("img",callback)
		$.each(arr,callback);                      
```



## 核心函数

### 介绍

```js
介绍
	jQuery向外暴露的就是jQuery函数, 可以直接使用
语法例子
	// $(document).ready()的简写
	$()
```

### 参数

```js
1. 接受一个函数
	$(function(){
        alert("hello");
    })
2. 接受一个字符串
	// 返回一个 jQuery 对象， 对象中保存了找到的DOM元素
    字符串选择器   	
        var $box = $("#box");
    代码片段
        // 会根据响应的代码片段创建对应的片段（DOM元素）
	    var $p = $("<p>hello</p>");
3. 一个 DOM 元素
	 // 会将原生 DOM 元素包装成一个 jQuery 对象
     // 注意这里是没有引号的
		var $div = $(div);
```

### 执行

```js
// 控制 jQuery 代码的执行
$.holdReady(hold);
hold = false  :  恢复 ready 函数的执行
hold = true :  暂停 ready 函数的执行
```
