## 数据类型

```ts
typescript 与 javacript 有着几乎相同的类型

// 	关键字 介绍
number		任意数字	
boolean	  	布尔值		
string		任意字符串

// 任意类型
any			任意类型
unknown		类型安全的any

// 对象
object		js 中的对象
enum		枚举 

//数组
array		js 数组
  			元组(tuple) ，固定长度数组

// 函数
void         没有值或者undefined
never	    永远不会返回结果

// 类型符号
|		联合类型
?		可选类型
&		同时
```





## 类型级别

```js

	number	boolean  string object arry tuple
一级子类	any unknown void 
二级子类	null  undefined
三级子类	never

不同级别
	never 可以赋值给其他类型，但是其他类型不能赋值给 never 类型
    
而在TypeScript中启用严格的空校验（--strictNullChecks）特性，
就可以使得null 和 undefined 只能被赋值给 void 或本身对应的类型
```





## 类型断言

```ts
介绍
	告诉解析器变量的实际类型 ： e 的类型为 string
    手动指定一个值的类型，即允许变量从一种类型更改为另一种类型
语法
	方法一 ：<Type>value
	方法二 ：值 as 类型
例子
	var str2:number = <number> <any> str   //str2 是 string 类型
	s = e as string;
```

## 类型别名

```ts
// string 的别名就是 newtype  , 两者都是一样的
type newtype = string
type newtype =  number|string|boolean
let a newtype  // a 可以是上面三种类型中的一个
```
