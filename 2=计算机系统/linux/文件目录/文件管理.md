## 文件

### 管理

```java
ls
    显示指定工作目录下的内容及属性信息
cp
    复制文件或者目录
mkdir
    创建目录
mv
    移动或者改名文件
pwd
    显示当前路径
```

### 编辑

```java
cat
    在终端设备上显示文件内容
echo
    输出字符串或者提取 shell 变量的值
rm
    移除文件或者目录
tail
    查看文件尾部内容
rmdir
    删除空目录
```

### 传输

```java
tftp
    上传和下载文件
curl
    文件传输工具
fsck
    检查并修复 linux 文件系统
ftpwho
    显示 ftp 会话信息
lprm
    删除打印队列中的打印任务
```
