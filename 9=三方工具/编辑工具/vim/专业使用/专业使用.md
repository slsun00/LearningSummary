## 打开文件

### 单文件

```java
命令
    vim [OPTION] 文件名字
    
注意
    存在： 文件被打开并显示内容
    不存在： 当编辑后第一次存盘时创建它
	目录不存在： 将无法保存编辑内容

参数
    +	光标在尾行行首。 // vim + test.txt
    +#	光标在第#行的行首。 // vim +3 text.txt    
    +/PATTERN	光标在第一个被PATTERN匹配到的行的行首  //  vim +/a  text.txt
    
    
    -b file 二进制方式打开
    -m file	只读打开，修改的内容不会被保存

    
    ex file或vim –e	直接进入扩展命令模式
    
    
    -d file1 file2…	比较多个文件
```



## vim 寄存器

```jav
有26个命名寄存器和1个无命名寄存器，常存放不同的剪贴版内容，可以不同会话间共享
寄存器名称
	a，b,…,z
格式：
	“寄存器放在数字和命令之间
例子	
	3"tyy 表示复制3行到t寄存器中
	"tp 表示将t寄存器内容粘贴
	未指定，将使用无命名寄存器

有10个数字寄存器，
	寄存器名称
		用0，1，…，9表示，
	作用
		0存放最近复制内容，
		1存放最近删除内容。
		当新的文本变更和删除时，1转存到2，2转存到3，以此类推。
		数字寄存器不能在不同会话间共享

```
