## 文本对象

```java
格式
    // 对符号内部文本的操作
    ci符号 // 更改，删除插入
    di符号 // 删除
    yi符号 // 复制
    vi符号 // 选中
    va符号 // 选中文本的是同，选中符号
操作符等
    c(change) d(delete) y（yaild） v(vioce)
范围    
    i  inside	内部串
    a   around	带符号的串
符号    
    成对符号
    	// vi[ 选中[]中间内容
		' " ` ( [ { <  
	其他组合
        t 选中中间的内容
        
        b 选中一个block
        w 选中一个单词
        p 选中一个段落
	    s 选中一个句子
    	
    	B 内含大块
   	   	W 内含字符串
    	
    
语法例子
    123"aa"456  
    需求： 删除双引号内部的 aa
    操作： 进入命令模式，将光标移动到双引号内，直接输入 di"
    结果: 123""456
```
