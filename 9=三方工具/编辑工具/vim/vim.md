## 模式综述

#### 作用

```java
1.命令(Normal)模式 ，默认模式，
    移动光标，
    剪切/粘贴文本。
2.输入模式
    修改文本。
3. 底线模式 
    文件保存，退出等
```

## 文档

```java
vimtutor 命令是以vim编辑器打开vim帮助文档，而且在里面做的修改不会保存。
在扩展命令模式下输入以下命令可以查看vim内置帮助：
help
helptopic
help option-list
setor:set all
```
