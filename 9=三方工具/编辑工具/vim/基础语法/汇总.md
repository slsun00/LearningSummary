## 作用

```java
1.命令(Normal)模式 ，默认模式，
    移动光标，
    剪切/粘贴文本。
2.输入模式
    修改文本。
3. 底线模式 
    文件保存，退出等
```



## 切换

```java
介绍
	所有模式切换都要经过命令模式，没有例外
    模式 a -- 命令模式 -- 模式 b
    

命令模式
    开启 vim 就是

     aio （哎哦）
底线命令行模式
    ：
其他模式
    替换模式 R
    可视模式 V
    
模式推出
    Esc键	   退出当前模式
    2Esc键	   总是返回到命令模式
	2ESC,enter	扩展命令模式转到命令模式
```
