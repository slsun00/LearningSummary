## 介绍

* 可以简单理解为指针

## 种类

| 种类 | 功能                                   | 支持运算                                 |
| ---- | -------------------------------------- | ---------------------------------------- |
| 输入 | 只读                                   | 只读， ++ 、==、!=                       |
| 输出 | 只写                                   | 只写， --                                |
| 前向 | 读操作<br>向前推进迭代器               | 读写  ++、==、 ！=                       |
| 双向 | 写操作<br>向前或者向后操作             | 读写  ++、--                             |
| 随机 | 读写操作<br>可以跳跃的方式访问任意数据 | 读写  ++、-- 、<、<= 、>、>=<br>[n] 、-n |
