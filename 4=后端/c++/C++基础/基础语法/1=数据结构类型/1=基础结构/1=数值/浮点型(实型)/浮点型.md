# 1.三要素

```
特点
	浮点数=符号位+指数位+尾数位

```



# 3.类型c

| 类型   | 精度   | 有效数字范围    | 占用储存空间 | 表述范围 |
| ------ | ------ | --------------- | ------------ | -------- |
| float  | 单精度 | 7位有效数字     | 4字节        |          |
| double | 双精度 | 15-16位有效数字 | 8字节        | 默认类型 |
