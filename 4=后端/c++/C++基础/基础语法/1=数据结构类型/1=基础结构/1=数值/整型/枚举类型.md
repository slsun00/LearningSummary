## 介绍

* enum(枚举)
* 枚举是 C 语言中的一种基本数据类型
* 它可以让数据更简洁，更易读

## 声明赋值

```c++
语法
    enum　枚举名　{枚举元素1,枚举元素2,……};
声明
    // 先定义枚举类型，再定义枚举变量
    enum DAY {  MON=1, TUE, WED};
    enum DAY day;
	// 定义枚举类型的同时定义枚举变量
	enum DAY {  MON=1, TUE, WED} day;

语法例子
    enum DAY {
    	MON = 1;
    	TUE ;  //2
    	FRI = 5 ; 
    	SUN  ; // 6
	}
  	int mian(){
        enum Day day;
        day = TUE ;
        printf(day);  // 6
        return 0;
    }
注意
    第一个枚举成员的默认值为整型的 0，后续枚举成员的值在前一个成员上加 1
    没有指定值的枚举元素，其值为前一元素加 1
    
```

## 操作

### 遍历

```c++
介绍
	C 语言中，枚举类型是被当做 int 或者 unsigned int 类型来处理的，
    所以按照 C 语言规范是没有办法遍历枚举类型的
特殊
    枚举类型连续的可以遍历 ：
    	enum DAY{ MON=1, TUE, WED, THU, FRI, SAT, SUN}
    不连续的不能遍历
         enum DAY{ MON=1, FRI=5, SAT, SUN}
```
