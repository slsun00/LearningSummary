## 运算符重载

## 介绍

```c++
对已有的运算符重新进行定义 ， 赋予另一种功能 ， 以适应不同的数据类型
```

## 加号重载

```c++
作用
    实现两个自定义类型的相加运算
编辑器符号
    operator+
方式
    成员函数重载
    	// Person p3 = p1.operator+(p2);
    	Person p3 = p1 + p2
    全局函数重载
    	// Person p3 = operator+(p1,p2);
    	Person p3 = p1 + p2
例子
 // 成员变量重载
    class  Person {
    public :
        // 成员函数实现重载
        Person operator+(Person &p){
            Person temp;
            temp.m_A = this->m_A + p.m_A;
            temp.m_B = this->m_B + p.m_B;
            return temp;
        }
        int m_A;
        int m_B;
    }
	void test01() {
        Person p1;
        p1.m_A = 10;
        p1.m_B = 10;
        
        Person p2;
        p2.m_A = 10;
        p2.m_B = 10;
        
        Person p3 = p1 + p2
    };

	// 	全局函数实现重载
	Person operator+(Person &p1, Person &p2){
        Person temp;
        temp.m_A = p1.m_A + p2.m_A;
        temp.m_B = p1.m_B + p2.m_B;
        return temp;
    }

```
