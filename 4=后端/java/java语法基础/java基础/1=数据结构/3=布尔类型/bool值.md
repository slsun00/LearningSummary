## 1. 基本介绍

```java
java 中的布尔值
    java 不允许其他类型来替代布尔值
    	比如：数字不允许作为一个布尔值使用
```



| 类型    | 值                             | 大小  |
| ------- | ------------------------------ | ----- |
| boolean | true (本质1) <br>false (本质0) | 1字节 |
