

## 常量

### 介绍

```java
格式
    // 编译时常量, 全局常量
    public static final 类型 变量名 = 变量值
 解释
    public 可用于包外
    static 只有一份
    final  说明是一个常量
命名
    变量名使用大写，用下划线分割
    public static final String A_NAME = "lili";
```

### 直接常量

```java
介绍
    直接常量后面的后缀字符标志了它的类型
    long f = 1L;
常见后缀
    long L
    float F
    double D
前缀
    十六进制 0X(或者0x) + 数字/字母
    八进制   0 + 数字
    二进制  没有直接常量表示方法
    
```
