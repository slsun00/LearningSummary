## 位运算符

### 介绍

```java
// 不使用短路
运算符         运算
&         按位与  双目运算，功能是参与运算的两位数各对应的二进制相与，规则：同时为1，结果为1
|         按位或  双目运算，功能是参与运算的两位数各对应的二进制相或，规则：一个位1，结果为1
^         按位非  双目运算，功能是参与运算的两位数各对应的二进制相非。规则：二进位不同，结果为1


int x = 1;
int y = 1;
//------------------------------------------
if (x++==2 & ++y==2) { // (false & true)
    x=7;
}
// x=2
// y=2

//-----------------------------------------
if (x++==1 | ++y==1) { // (true & false)
    x=7;
}
// x=7
// y=2
```

## 位移运算符

### 介绍

``` java
注意
    参与计算的数，是转化为 二进制的数进行计算的
    
<<        左移， 双目运算，
    把<<左侧的运算数的各二进位全部左移若干位，
    被移除的高位丢弃，空缺位(低位)补0，
    一定范围内： 左移n位就是乘以2的n次方
>>        右移    双目运算，
    把>>左侧的运算数的各二进制全部右移若干位，
    被移位
    	二进制最高位是 0 ，右移后空缺位补 0
    	二进制最高位是 1 ， 右移后空缺位补 1
    一定范围内： 右移n位就是除以2的n次方
    
>>> 	  无符号右移
    被移位二进制最高位无论是 0 或者 1 ，空缺位都用 0 补充
    
    
// 备注：就把二进制的补码，被冻结成尺子，让整个尺子进行》箭头的方向移动，对，就是整体移动
1.右移运算符：低位溢出，符号位不变，并用符号位补溢出的高位
2.左移运算符：符号位不变，低位补0

注意
	5>>2  或者 5<<2  计算都是将 5 变为二进制，然后进行位运算
    
例子
    3 << 3 = 24  等价于  3 * 2 * 2 * 2 = 3 乘以 2 的 3 次方
    3 >> 1 = 1 等价于  3 / 2  即 3 除以 2 的 1 次方
    
注意
    1. 位移操作的都是整型的数据
    2. 在一定范围内：(正负都一样)
    	// 左加右减
    	向左移动一位， 相当于 * 2
    	向右移动一位， 相当于 / 2
```
