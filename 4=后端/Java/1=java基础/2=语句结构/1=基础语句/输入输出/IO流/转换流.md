## 转换流

## 介绍

* 编码
    * 按照某种规则，将字符存储到计算机中
    * 字符 -->  字节
* 解码
    * 按照某种规则，见计算机中的二进制数据显示出来
    * 字节 --> 字符
* 字符集
    * charset  编码表

## 原理

```java
java.io.InputStreamReader
    是 reader 的子类，是字节流到字符流的桥梁
    读取字节，并用指定的字符集将其解码为字符
    字符集可以有名称指定、也可以使用默认的
    

```
