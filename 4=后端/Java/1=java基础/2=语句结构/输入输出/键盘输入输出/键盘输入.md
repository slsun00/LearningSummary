   * InputStreamReader和BufferedReader方法 
   * 优点: 可以获取键盘输入的字符串 
   * 缺点: 如何要获取的是int,float等类型的仍然需要转换

  public static void ReadTest(){ 
    System.out.println("ReadTest, Please Enter Data:");

    try{ //该方法中有个IOExcepiton需要捕获 
      String name = br.readLine();

  }
