## 介绍

* 包装类都是引用类型
* 位于 java.lang 包中

## 基础包装类

```java
数值
    整型
    	byte	Byte
    	short	Short
    	int		Integer [特殊]
    	long	Long
    浮点
    	float	Float
    	double	Double
    字符
    	char	Character [特殊]
    布尔
    	boolean	Boolean
```

## 其他类

```java
String
```



## 介绍

```java
从 jdk1.5 开始， 支持 自动装箱、自动拆箱
    自动装箱 ： 基本类型 --> 包装类型
    自动拆箱 ： 包装类型 --> 基本类型
```
