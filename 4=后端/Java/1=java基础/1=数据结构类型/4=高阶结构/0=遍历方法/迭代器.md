## 实现

```java
迭代器是一个接口， 实现方式比较特殊，
利用 Collection 接口中有一个方法叫 iterator() 方法返回的就是迭代器的实现类对象
 // 使用
 集合页中的方法 itrator() 获取迭代器的实现类，多态使用 Iterator // 集合是什么类型，迭代器就是什么类型
    hasNext 判断有无下一个元素， 
    next 取出下一个元素
```



## 方法

```java
hasNext()
next
```

## 例子

```java
Collection<string> coll = new ArrayList<>();
Iterator <String> it = coll.iterator();
while (it.hasNext()){
    String e = it.next();
    System.out.print(e);
}
```
