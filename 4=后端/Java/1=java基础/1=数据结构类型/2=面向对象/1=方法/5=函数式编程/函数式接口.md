## 概念

* 有且仅有一个抽象方法的接口
* 可以适用于 Lamba 使用的接口

## 格式

```java
语法
    修饰符 interface 接口名称 {
        public static 返回值类型 方法名称(可选参数)；
        // 其他非抽象方法
    }
注解
    格式
    	@FunctionalInterface
 	作用
    	检测接口中是否只有一个函数式接口，
```



## 使用

```java
介绍
    一般作为方法的参数、返回值类型
 使用例子
    @FunctionalInterface
    public interface A {
    	void get()；
	}
	public class AImpl implements A{
        
    }
	class Demo {
        public static void show(A a){ a.get(); }
        public static void main(String[] args){
			// 使用实现类
            show(new implements);
            // 使用匿名内部类
            show(new A(){
                @Override
                get(){Systerm.out.println("1123")}
            });
            // 简写
            show(()->{
                Systerm.out.println("1123");
            });
        }
        
    }
```

### Supelier

```java
介绍
    java.io.function.Supplier<T> 接口
    生产型接口，指定接口的泛型是什么类型， 那么接口中的 get 方法就会生产什么类型的数据
public class test2 {
    public static String getString(Supplier<String> sup){
        return sup.get();
    }

    public static void main(String[] args) {
        String s = getString(()->{
            // 重写方法体
            return  "胡歌";
        });
        System.out.println(s); // 胡歌
    }
}
```
