## 介绍

```java
关键字
    enum {}
    
介绍
    java se5 添加的
	它可以让数据更简洁，更易读
本质
    是一个类，有自己的方法
    将一组具名的值的有限集合创建为一种新的类型，这些具名的值可以作为常规的程序组件使用
```



## 声明赋值

```c++
语法
   修饰符 enum　枚举名　{枚举元素1,枚举元素2,……};
声明
    // 先定义枚举类型，再定义枚举变量
    enum DAY {  MON=1, TUE, WED};
    enum DAY day;
	// 定义枚举类型的同时定义枚举变量
	enum DAY {  MON=1, TUE, WED} day;

语法例子
    public enum DAY {
    	MON = 1;
    	TUE ;  //2
    	FRI = 5 ; 
    	SUN  ; // 6
	}
	// 使用
        enum Day day;
        day = TUE ;
注意
    // 只要有成员(语句)就会加 1
    第一个枚举成员的默认值为整型的 0，后续枚举成员的值在前一个成员上加 1
    没有指定值的枚举元素，其值为前一元素加 1
    
```

## 操作

### 创建

```java
创建 enum 时， 编译器会为你生成一个相关的类，该类继承 java.lang.Enum
```



### 使用 

```java
// 在 switch 中使用
enum 中的名字可以用于 case 中
```



### 遍历

```c++
介绍
	enum 的 values() 方法，可以遍历 enum 实例，
    返回 enum 实例的数组， 该数组会严格保持声明的时候的顺序
特殊
    枚举类型连续的可以遍历 ：
    	enum DAY{ MON=1, TUE, WED, THU, FRI, SAT, SUN}
    不连续的不能遍历
         enum DAY{ MON=1, FRI=5, SAT, SUN}
```

### 比较

```java
可以使用 == 来比较 enum 实例， 
    编译器会自动 equals() hashCode() 方法
    实现了 Comparable 、Serializable 接口
```

### 查询

```java
name() 返回 enum 实例声明时的名字， 等价于 toString
valueOf() 给定名字，返回响应的  enum 实例
```
