## 介绍

```java
关键字
    enum {}
    
介绍
    java se5 添加的
	它可以让数据更简洁，更易读
本质
    是一个类，有自己的方法
    将一组具名的值的有限集合创建为一种新的类型，这些具名的值可以作为常规的程序组件使用
```

### 枚举实例成员

```java

public enum MyDay {
   
    private int code;
    private String name;
    
    // enum实例序列
    MONDAY(1,"星期一"),
    THUSDAY(2,"星期二");//这个后面必须有分号
    
    // 构造函数只能使用 private 访问修饰符，所以外部无法调用
    // 而且没有返回值
    private MyDay(int code,String name) {
        this.code = code;
        this.name = name();
    }
    
    // 其set get 方法，注意是针对上面的字段的，不是实例
}


注意
   	1. 如果枚举类具有抽象方法，则枚举类的每个实例都必须实现它。    
    2. 枚举类中的抽象方法实现，需要枚举类中的每个对象都对其进行实现。
    3. 枚举类型中可以有静态方法，也可以与其他方法。可以有属性与get，set方法
    4. 可以 覆盖方法
```

### 继承

```java
介绍
    所有的枚举都继承自java.lang.Enum类。
    由于Java 不支持多继承，所以枚举对象不能再继承其他类
    但是可以实现接口
    
    
使用接口管理枚举类
    public interface Food {  
        enum Coffee implements Food{  
            BLACK_COFFEE,DECAF_COFFEE,LATTE,CAPPUCCINO  
        }  
        enum Dessert implements Food{  
            FRUIT, CAKE, GELATO  
        }  
    }  

```

## 操作

### 创建

```java
创建 enum 时， 编译器会为你生成一个相关的类，该类继承 java.lang.Enum
```





### 遍历

```c++
介绍
	enum 的 values() 方法，可以遍历 enum 实例，
    返回 enum 实例的数组， 该数组会严格保持声明的时候的顺序
特殊
    枚举类型连续的可以遍历 ：
    	enum DAY{ MON=1, TUE, WED, THU, FRI, SAT, SUN}
    不连续的不能遍历
         enum DAY{ MON=1, FRI=5, SAT, SUN}
```

### 比较

```java
可以使用 == 来比较 enum 实例， 
    编译器会自动 equals() hashCode() 方法
    实现了 Comparable 、Serializable 接口
```

### 查询

```java

valueOf()
    	给定名字，返回响应的  enum 实例
```
