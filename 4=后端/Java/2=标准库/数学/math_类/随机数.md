## 随机数种子

```java
介绍
    计算机中的随机数是通过一系列算法得出来的，并不是真正的随即数
    种子数字：就是你提供给计算机一个数字，计算机基于这个数来计算出不同的值
注意
    种子数只是随机算法的起源数字，和生成的随机数的区间没有任何关系
例子
    // 平方取中法
    // 你提供给计算机一个数，作为初始条件，计算机根据这个数得出来不同的值
    1）将种子设为X0，并mod 10000得到4位数
    2）将它平方得到一个8位数（不足8位时前面补0）
    3）取中间的4位数可得到下一个4位随机数X1
    4）重复1-3步，即可产生多个随机数
```

## Random

### 基础

```java
介绍
    是一个伪随机数，有规则的随机，所谓有规则的就是在给定种(seed)的区间内随机生成数字
    相同种子数的Random对象，相同次数生成的随机数字是完全相同的；
    种子数只是随机算法的起源数字，和生成的随机数的区间没有任何关系
    生成的随机数字都是均匀分布的，也就是说区间内部的数字生成的几率均等
构造函数
    默认值：以当前系统时间为种子
    
```

### 常用例子

```java
[0,n)      范围的        new Random().nextInt(n)
[m n] 范围的(闭区间)   new Integer ((int)(Math.random()*(n - m + 1) + m))
```
