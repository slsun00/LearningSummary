## math_类

* 数学相关的工具类，提供大量的静态方法
* 完成数学计算

## 常用方法

```java
public static double abs(double num);  // 去绝对值多种重载
public static double ceil(double num);  // 向上取整
public static double floor(double num);  // 向下取整
public static long round(double num);  // 四舍五入

// 使用
math.ceil(3.14) // 3.14
math.round(3.1) // 3
    
// 常量
PI  // 圆周率  math.PI;
```
