## 发展

* 1990年开发 oak , 后改名为 Java
* 1996年发布 JDK1.0
* 2000年 ， jdk1.3 ， Java HotSpot Virtual Machine 正式发布 ， 称为 Java 默认的虚拟机
* 2002年， jdk1.4 , classic虚拟机退出
* 2004年 ， jdk1.5 , 改名为 Java5.0
