

## 基础

```go
[...]  字符集（字符类），
```



## 1.单字符串

```
.              任意字符（标志s==true时还包括换行符）
[xyz]          字符族,对应的位置可以是字符集中任意字符。
[^xyz]         反向字符族
\d             Perl预定义字符族
\D             反向Perl预定义字符族
[:alpha:]      ASCII字符族
[:^alpha:]     反向ASCII字符族
\pN            Unicode字符族（单字符名），参见unicode包
\PN            反向Unicode字符族（单字符名）
\p{Greek}      Unicode字符族（完整字符名）
\P{Greek}      反向Unicode字符族（完整字符名）
```

## 2.组合

```
xy             匹配x后接着匹配y
x|y            匹配x或y（优先匹配x）
```

## 3.重复

```go
x			  匹配自身，一般字符
//--------------------------------------------------------------------------
x*             匹配前一个字符(x) 0 或无限次，越多越好（优先重复匹配x）
x+             匹配前一个字符(x) 1 次或无限次，越多越好（优先重复匹配x）
x?             0或1次匹配x，优先1次
x{m,n}		  匹配前一个字符 m 至 n 次 ,越多越好
x{m,}          重复>=m次匹配x，越多越好（优先重复匹配x）
x{n}           则匹配前一个字符(x) 0 至 n 次 ,越多越好

//-----------------------------------------------------------
以上在后面加一个 ？ ，表示匹配次数越少越好

```

## 4.分组

```
(re)           编号的捕获分组
(?P<name>re)   命名并编号的捕获分组
(?:re)         不捕获的分组
(?flags)       设置当前所在分组的标志，不捕获也不匹配
(?flags:re)    设置re段的标志，不捕获的分组
```

## 5.边界匹配

```
字符串 文本

^       匹配字符串开头，在多行模式中匹配每一行的开头；匹配文本开始，标志m为真时，还匹配行首
$       匹配字符串末尾，在多行模式中匹配每一行的末尾；匹配文本结尾，标志m为真时，还匹配行尾
\A      仅匹配文本开始
\b      单词边界（一边字符属于\w，另一边为文首、文尾、行首、行尾或属于\W）
\B      非单词边界
\z      匹配文本结尾
```

## 6.转义序列

```
\			 转义字符
\a             响铃符（\007）
\f             换纸符（\014）
\t             水平制表符（\011）
\n             换行符（\012）
\r             回车符（\015）
\v             垂直制表符（\013）
\123           八进制表示的字符码（最多三个数字）
\x7F           十六进制表示的字符码（必须两个数字）
\x{10FFFF}     十六进制表示的字符码
\*             字面值'*'
\Q...\E        反斜线后面的字符的字面值
```

## 7.字符族

*  预定义字符族之外，方括号内部 

```
 x             单个字符 ，字符族本身
A-Z            字符范围（方括号内部才可以用）
\d             Perl字符族
[:foo:]        ASCII字符族
\pF            单字符名的Unicode字符族
\p{Foo}        完整字符名的Unicode字符族
```

## 8. 预定义字符族作为字符族的元素： 

```
[\d]           == \d
[^\d]          == \D
[\D]           == \D
[^\D]          == \d
[[:name:]]     == [:name:]
[^[:name:]]    == [:^name:]
[\p{Name}]     == \p{Name}
[^\p{Name}]    == \P{Name}
```

## 9.perl 字符族

```
\d              [0-9]
\D              [^0-9]
\s              [\t\n\f\r ]
\S              [^\t\n\f\r ]
\w              [0-9A-Za-z_]
\W              [^0-9A-Za-z_]
```

# 10 ASCII字符族

```
[:alnum:]      == [0-9A-Za-z]
[:alpha:]      == [A-Za-z]
[:ascii:]      == [\x00-\x7F]
[:blank:]      == [\t ]
[:cntrl:]      == [\x00-\x1F\x7F]
[:digit:]      == [0-9]
[:graph:]      == [!-~] == [A-Za-z0-9!"#$%&'()*+,\-./:;<=>?@[\\\]^_`{|}~]
[:lower:]      == [a-z]
[:print:]      == [ -~] == [ [:graph:]]
[:punct:]      == [!-/:-@[-`{-~]
[:space:]      == [\t\n\v\f\r ]
[:upper:]      == [A-Z]
[:word:]       == [0-9A-Za-z_]
[:xdigit:]     == [0-9A-Fa-f]
```
